# Açaí Delícias do Pará - Gestão de Caixa

Este é o seu sistema de frente de caixa (PDV) moderno, desenvolvido com Next.js e Firebase.

## 🔗 Links do Projeto

- **Link de Desenvolvimento:** [Ambiente atual no Cloud Workstations]
- **Link Permanente (Produção):** [https://nextn-f5a13.web.app](https://nextn-f5a13.web.app)
- **Console do Firebase:** [https://console.firebase.google.com/project/nextn-f5a13](https://console.firebase.google.com/project/nextn-f5a13)

## 🚀 Como colocar no ar (Deploy)

Para que o link permanente funcione e você possa acessar o sistema de qualquer lugar:

1. Acesse o [Console do Firebase](https://console.firebase.google.com/project/nextn-f5a13).
2. No menu lateral, vá em **Build > App Hosting** ou **Hosting**.
3. Siga as instruções para conectar seu repositório do GitHub.
4. O Firebase irá gerar automaticamente o link `nextn-f5a13.web.app`.

## 🛠️ Configurações Necessárias

Certifique-se de ativar no console:
- **Firestore Database:** No modo "Produção" ou "Teste".
- **Authentication:** Ative o provedor "Anônimo" (Anonymous) para permitir o uso sem login obrigatório imediato.

## ✨ Funcionalidades
- Gestão de Entradas (PIX, Cartão, Dinheiro, Delivery).
- Gestão de Saídas (Sangria e Despesas).
- Insights com IA para categorização de gastos.
- Dashboard financeiro em tempo real.
- Impressão de relatórios formatados.

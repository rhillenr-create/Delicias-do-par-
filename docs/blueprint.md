# **App Name**: AçaíLume Pro

## Core Features:

- High-Contrast Quick Action POS: A touchscreen-optimized dashboard featuring oversized action buttons for PIX, Card, Delivery, and Cash transactions.
- Live Ledger Engine: Persistent real-time data storage for all incoming and outgoing movements powered by Firestore integration.
- AI Insight Tool: An AI assistant that analyzes transaction descriptions to automatically categorize expenses and suggest potential cost-saving measures based on spending patterns.
- Dynamic Flow Explorer: Interactive listing of movements with advanced search, period filters (day, week, month), and inline CRUD editing capabilities.
- Luminous Analytics Dashboard: Visual breakdown of revenue split by payment type (PIX, Credit, Debit) vs total expenses and net profit calculations.
- Instant Monthly Reporter: A specialized tool that compiles full monthly financials into a print-ready PDF layout for easy physical archival.
- Local Cache Resilience: Service worker implementation to ensure core transaction logging works offline during connectivity drops in the store.

## Style Guidelines:

- Primary color: Vibrant Amethyst Purple (#9F33FF) symbolizing the açaí berry, used for active states and primary branding.
- Background: Midnight Obsidian (#09070A), a deep desaturated purple-black that enhances readability for neon elements.
- Accent: Deep Indigo (#8080FF) for UI components, complemented by Neon Green (#39FF14) strictly for successful totals and primary call-to-action highlights.
- Headline and body font pairing: 'Space Grotesk' for technical headings and 'Inter' for highly legible transaction data and lists.
- A mobile-first grid system using high-profile button cards with distinct touch targets for fast-paced retail environments.
- Fluid 'neon-glow' pulse effects when capturing new movements and soft page transitions between the Cashier and Movements tabs.
- Clean, thin-line glyphs from a tech-inspired set to denote different payment methods and movement types.
